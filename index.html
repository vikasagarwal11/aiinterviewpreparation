Privacy Rule

This public version teaches concepts and interview frameworks. It should not contain your actual product internals, exact system architecture, agent decomposition, schemas, prompts, patent-sensitive ideas, customer data, or private employer details.

Safe

Generic examples, public concepts, broad diagrams, interview questions, and non-proprietary analogies.

Private

Exact architecture, named internal workflows, unique rules, private data, IP-sensitive features, and real story details.

Reality

noindex is not security. Use Render auth/password, private repo, or offline files for truly private content.

Deploy Instructions

  1. Replace repo root files with this package.
  2. Ensure assets/styles.css and assets/app.js exist.
  3. Delete old root-level styles.css and script.js.
  4. Commit and push to main.
  5. Render redeploys automatically.
  6. After deploy, search for “RAG”, open a lesson, and check browser console for errors.
